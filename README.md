# NIGHT OWL 🌙  
A Simple and Elegant Login Interface

NIGHT OWL is a clean and modern login page designed with a dark, night‑friendly theme. It can be easily integrated into any website or application that requires user authentication.

## Features ✨ 
- Minimal and intuitive UI  
- Fully responsive layout for all screen sizes  
- Easy to integrate with any backend or authentication system  

## Tech Stack 🛠
- **HTML5**  
- **CSS3**  
- **JavaScript**
